启动参数

–xformers 使用xformers库。极大地改善了内存消耗和速度。Windows 版本安装由C43H66N12O12S2 维护的二进制文件
–force-enable-xformers 无论程序是否认为您可以运行它，都启用 xformers。不要报告你运行它的错误。
–opt-split-attention Cross attention layer optimization 优化显着减少了内存使用，几乎没有成本（一些报告改进了性能）。黑魔法。默认情况下torch.cuda，包括 NVidia 和 AMD 卡。
–disable-opt-split-attention 禁用上面的优化
–opt-split-attention-v1 使用上述优化的旧版本，它不会占用大量内存（它将使用更少的 VRAM，但会限制您可以制作的最大图片大小）。
–medvram 通过将稳定扩散模型分为三部分，使其消耗更少的VRAM，即cond（用于将文本转换为数字表示）、first_stage（用于将图片转换为潜在空间并返回）和unet（用于潜在空间的实际去噪），并使其始终只有一个在VRAM中，将其他部分发送到CPU RAM。降低性能，但只会降低一点-除非启用实时预览。
–lowvram 对上面更彻底的优化，将 unet 拆分成多个模块，VRAM 中只保留一个模块,破坏性能
*do-not-batch-cond-uncond 防止在采样过程中对正面和负面提示进行批处理，这基本上可以让您以 0.5 批量大小运行，从而节省大量内存。降低性能。不是命令行选项，而是使用–medvramor 隐式启用的优化–lowvram。
–always-batch-cond-uncond 禁用上述优化。只有与–medvram或–lowvram一起使用才有意义
–opt-channelslast 更改 torch 内存类型，以稳定扩散到最后一个通道,效果没有仔细研究。
–use-cpu 只使用CPU

–use-cpu注意事项

低显存使用NovelAI的方法

在stable-diffusion-webui文件夹下找到webui-user.bat，其内容如下：

@echo off

set PYTHON=

set GIT=

set VENV_DIR=

set COMMANDLINE_ARGS=

call webui.bat

在上述文件中的“set COMMANDLINE_ARGS=”后根据自己显卡的显存选择下面的参数填入

4G显存：–precision full --no-half --lowvram --always-batch-cond-uncond

6G显存：–precision full --no-half --medvram

低于5G显存：–medvram

低于3G显存：–lowvram --always-batch-cond-uncond

–precision full --no-half --lowvram --medvram

  1. 关于Novel AI — tag串的编写以及一些基本概念

    TAG

    定义：也就是关键词，由正面tag 和 反面tag 相互作用产生结果。

    逻辑：AI会优先识别相对靠前的 tag关键词，类似英文的逻辑（或许）。

    排版：学会分段描述各部分tag串，就好比 分图层 一样，将有助于你更好的区分和修改各部分环节。

    中心设定（依顺序）

    前置tag串(做的事情or姿势动作) ——中间(人设)——最后(背景)。

    人设拆解

    人设：依照顺序：职业设定—身体设定—服装设定—饰品设定 的描述。

    身体：表情—发型—瞳孔—眼睛—面部—纹身—皮肤—身材—特征(是否有翼、尾巴、猫耳 或长角)。

    服装：衣服—裙子or裤子—手套—鞋—袜—帽子。

    饰品：发花or发带，皇冠，蝴蝶结，领带，戒指，耳环，项链 等。

    完整优化（依顺序）

    画质串——模型——主画风——时代和风格氛围——色彩搭配——光影——运镜——中心设定

    拆解

    画质：杰作，由著名艺术家，高清，高清壁纸，详细的CG unity 8k壁纸，非常精致和美丽的细节，锐化。 可以全写。

    模型：pt文件，放在安装文件夹的embedding里自动加载，可以替换原有的画风 或者人物的模型。想加载哪个模型，就写哪个文件名，文件改名之后 写更改后的名字。

    主画风：插图，游戏CG，4kCG，官方艺术，肖像，概念设计，水墨，水彩，油画，素描 等。想要混用哪些效果就写什么。也可以添加著名画家的风格，格式：by Name 。如果是单人图 最后加上 solo。

    时代和风格氛围：中世纪，现代，科幻 等 ——神圣的，庄严的，壮丽的 感觉 等——洛丽塔，哥特式，印第安，埃及风，中国风，赛博朋克 等

    色彩搭配：渐变颜色，色彩鲜艳的，金、银、红、绿、蓝、紫、青、宝石色、玻璃色、彩虹色 等。

    光影：光线追踪，电影灯光，神光，逆光，镜头闪光，戏剧性的阴影，极黑的阴影，镜面反射，玻璃反射 等。

    运镜：近or中or远景别，局部特写，人物焦点，鱼眼镜头，电影视角，动态视角，倾斜视角，景深，透视 等。

    动作：动态姿势，躺、坐、跪、站、蹲的姿势，鸭子坐，看着观众。

    人设：萝莉，剑客，JK，骑士，公主，忍者，新娘 等。

    优化表情：开心的，难过的，愤怒的，哭泣的 等。

    优化头发：漂浮的，蓬松的，渐变色的——长发，短发，卷发，丸子头，双马尾 等。

    优化眼睛：美丽的，精致的，放大的，水一般的，有眼泪的 等。前两个tag在各地方通用。

    优化身材：锁骨，娇小的，平胸的，露肩的，露背的等。

    优化服装：xx上有褶皱，xx的质感，详细的，透明的，长or短袖的，浸湿的，蕾丝边的，撕裂的 等——长or短裙，连衣裙，百褶裙，婚纱，汉服，金属板甲 等——JK鞋，lolita鞋，婚鞋，长筒靴 等——黑白丝，连裤袜，吊带袜，小腿袜，过膝袜，大腿袜 等。

    优化饰品：羽毛装饰，金银制的or镶边的，在xx上镶钻，镶宝石等。

    优化背景：清晰而复杂的背景，模糊的背景，季节，清晨，黄昏，夜晚，山川，河流，森林，花朵，风，潮汐，海浪，漩涡，云，天空，草地，焦土，水，火，冰，龟裂，结晶，燃烧，雾化，爆炸，飞溅 等

    特殊背景：龙的背景，浮空的城堡，树屋，教堂，宫殿，战场 等。

    强调

    如果出图时 AI忽略了你想表达的tag (数值来源有待考证)

    直接在 tag 后加 冒号+数字 提升强调程度，如 1girl：1.1

    可以给 tag 加括号，如 (1girl），每个括号提升1.1的强度，两个1.2 ((1girl))，以此类推。

    反之，每个中括号 [1girl] 降低1.1的强度。

    大括号{1girl}也可以提升强度，具体数值暂且未知，但确实可以。

    句式表达

    用于强调及区分主体和次体，或因果关联。可实现单一关键词无法实现的效果。

    在中心设定中，如果主体是“一个椅子上的女孩”，房间是次要的背景。那么椅子也得 跟人物放在一起，也就是放在 动作和背景 的中间：

    动作——“一个椅子上的女孩”——背景

    同时在 次体背景的段首前 加 Background，用以前后区分（类似于分图层）。

    需要强调它是结合在一起的，可以直接写一个 同时包含人物、动作及物件 的句子，比如：

    “A girl sitting in a chair” 。如果画面中还是没有出现，则继续 给整个句子加括号强调表达。

    同理，在使用形容词、副词等，深入细化想要描述的人物和背景时，也可以直接写:

    “穿着破旧盔甲的战士正在与冷酷的敌人战斗”、“圣洁的女孩沐浴在精灵森林的泉水之中”、“美丽的火烧云在细致的天空上”、“破败的房屋之下埋藏了一本神秘的书”。等

    科技的提升是必然的结果，如果有画师因为受到资本冲击即将面临失业，那说明你还不够卷，顶级的画师是不会被AI淘汰的，甚至会利用它成为辅助作画的工具。与其打压AI，不如深造自己。
